# Bank API (TypeScript + Express + SQLite)

- Accounts + activities
- Activities: draw, deposit, loan
- Loans include interest (%) and payments.
- Validation with Joi in model files.

## Scripts

- `npm install` - install dependencies
- `npm run start` - start the API server (ts-node)

Server runs by default on: http://localhost:3000
API base path: `/api`.
